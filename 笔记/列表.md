# 🐍 Python 列表 (List) 核心学习笔记

## 🧭 1. 基础认知与切片 (查)
列表是有序的数据集合，可以通过索引（从 `0` 开始）和切片来提取内容。

### 1.1 常用切片语法
- `lst[0]`：获取第一个元素。
- `lst[0:3]`：提取索引 0 到 2 的元素（顾头不顾尾）。
- `lst[0:3:2]`：提取索引 0 到 2 的元素，步长为 2（隔一个拿一个）。
- `len(lst)`：获取列表的总长度。

## 🛠️ 2. 列表的增、删、改
下面是日常开发中最常用的列表操作方法总结：

| **操作** | **代码示例** | **核心作用** |
| :--- | :--- | :--- |
| **追加 (增)** | `lst.append("baigao")` | 在列表最末尾增加 1 个元素 |
| **插入 (增)** | `lst.insert(1, "xiao ming")` | 在指定索引（如 1）强行插队 |
| **扩展 (增)** | `lst.extend(["a", "b"])` | 批量将新列表的元素拼接到末尾 |
| **按位删 (删)**| `ret = lst.pop(3)` | 踢走指定索引的元素，可赋值给变量备用 |
| **按值删 (删)**| `lst.remove("xiao ming")` | 删除从左到右找到的第一个匹配项 |
| **赋值 (改)** | `lst[2] = "hihi"` | 直接覆盖指定索引位置的内容 |
| **排序** | `lst.sort()` | 对原列表进行永久性排序 |

*注：`append` 和 `extend` 的区别在于，前者把传入的内容当成一个整体，后者会把传入的列表打散后逐个添加。*

## 🔄 3. 列表的遍历与修改
根据不同的需求，选择不同的 `for` 循环方式。

### 3.1 仅读取数据
直接遍历列表元素，适合打印或条件判断：
```python
for item in lst:
    print(item)
```

### 3.2 遍历并修改原列表
必须通过 `range(len())` 获取索引，才能精准修改原位置的数据：
```python
for i in range(len(lst)):
    if lst[i].startswith("xiao"):
       lst[i] = "bai" + lst[i][4:] # 修改并放回原位
```

## 🪆 4. 嵌套列表的提取 (多维列表)
处理列表套列表的情况，核心法则是：**从外到内，逐层加中括号**。

```python
lst.insert(0, [114, 514, [1991, "baigao"]])

# 逐层剥洋葱：
# 第一层：lst[0] -> [114, 514, [1991, "baigao"]]
# 第二层：lst[0][2] -> [1991, "baigao"]
# 第三层：lst[0][2][1] -> "baigao"

# 提取并修改：
lst[0][2][1] = lst[0][2][1].upper()
```

## 🚨 5. 核心避坑：安全地循环删除
**严禁**在遍历列表的同时直接 `remove` 元素，会导致元素前移漏查。必须使用“临时列表”作为过渡。

### 5.1 标准安全删除模板
```python
temp = [] # 1. 准备临时列表（记仇本）

# 2. 找出需要删除的目标，存入临时列表
for i in lst:
    if i.startswith("bai"):
        temp.append(i)

# 3. 遍历临时列表，去原列表中执行删除（注意缩进与上一个 for 平齐）
for j in temp:
    lst.remove(j)
```
