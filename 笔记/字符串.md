# Python 基础数据类型详解：字符串 (String)

> **💡 核心铁律**：字符串是**不可变数据类型**。所有的字符串操作，都不会改变原字符串，而是会生成并返回一个新的字符串。

---

## 1. 核心基础：格式化、索引与切片

### 1.1 字符串格式化 (f-string)
目前 Python 中最推荐的字符串拼接方式，语法简洁高效：
```python
name = "Python"
print(f"Hello, {name}!")  # 输出: Hello, Python!
```

### 1.2 索引 (Indexing)
字符串里的字符是有固定位置的，**从 0 开始计数**。
- `s[0]`：获取第一个字符。
- `s[-1]`：获取最后一个字符（倒数第一）。

### 1.3 切片 (Slicing)
用于提取字符串中的一部分，语法规则为 `s[start:end:step]`（起始位置 : 结束位置 : 步长）。
- ⚠️ **顾头不顾尾**：`end` 位置的数据永远拿不到。
- **示例**：`s[0:3]` 只能拿到索引 0、1、2 的字符。

---

## 2. 字符串常用内置方法

日常开发中最常用的 7 个字符串处理方法速查表：

| **方法 / 函数** | **核心功能** | **常见应用场景** |
| :--- | :--- | :--- |
| `upper()` | 将字符串全部转为大写 | 验证码比对时忽略大小写 |
| `strip()` | 去除两端的空白字符 | 清理用户输入时多打的空格、换行符(`\n`)或制表符(`\t`) |
| `replace()` | 替换字符串中的指定内容 | 敏感词过滤，如把脏话替换为 `***` |
| `split()` | 按照指定字符切割字符串 | 将一段长文本按逗号切分，**返回一个列表 (List)** |
| `join()` | 将列表拼接成新字符串 | `split()` 的反向操作，把列表里的词重新连成一句话 |
| `startswith()` | 判断是否以指定内容开头 | 检查网址是否以 `http` 开头（同理有 `endswith()`） |
| `len()` | 获取字符串长度 | 检查密码长度是否达标（注意：它是内置函数，非字符串专属方法） |

---

## 3. 字符串的循环与遍历

关键字 `in` 在字符串处理中有两大核心妙用：

### 3.1 配合 `for` 循环遍历
可以把字符串想象成一列火车，逐个提取字符：
```python
s = "Hello"
for c in s:
    print(c)  # 依次打印 H, e, l, l, o
```

### 3.2 作为条件判断
直接判断“某个子串是否包含在长字符串中”，返回布尔值 (`True` / `False`)：
```python
if "Python" in "I love Python coding":
    print("关键字存在！")
```
