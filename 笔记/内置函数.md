# 🐍 Python 核心内置函数与二级考试通关指南

> **📝 导语**
> 本笔记结合了 Python 底层运行机制（哈希、布尔逻辑）与全国计算机等级考试（NCRE）二级 Python 的高频考点。
> 核心编程理念：数据结构负责**存储与查询**，格式化工具负责**排版与展示**。

---

## ⚖️ 1. 逻辑判断与真假鉴定

在 Python 中，`bool` 是程序做选择题的唯一依据。

### 核心法则：万物皆可 bool
- **`False` 阵营**：只要是“空的”（如 `""`, `[]`, `{}`）或者“零”（如 `0`, `0.0`），以及特殊值 `None`。
- **`True` 阵营**：只要“有东西”（如 `"白"`, `[1, 2]`, `-5`）。

### 相关内置函数
- **`bool(obj)`**：万能测谎仪，直接鉴定对象的真假。
- **`any(iterable)`**：逻辑扫描仪。只要可迭代对象中有一个元素为 `True`，即返回 `True`。（例如 `any([0, "", ''])` 全是空值，返回 `False`）。

## 🕵️‍♂️ 2. 数据结构与底层机制

理解底层机制是写出高效代码的关键，也是二级考试选择题的常客。

### 极速查找的灵魂
- **`hash(obj)`**：数字指纹提取机。将**不可变**的数据（如字符串、整数、元组）压缩成唯一整数。注意：**列表和字典不可哈希**。
- **`id(obj)`**：内存身份证。返回对象在内存中的唯一真实地址。
- **`type(obj)`**：血统鉴定书。返回变量的数据类型（如 `int`, `str`, `list`）。

### 核心容器转换
- **`list()` / `tuple()`**：列表与元组转换器。
- **`set()`**：无情的去重机器。自动剔除重复元素，内部绝对无序。
- **`dict()`**：高效的信息档案室。通过键值对（Key-Value）存储，键必须可哈希。

---

## 🔄 3. 循环、序列与迭代神器

处理列表和控制循环时最常打交道的好帮手。

### 遍历与生成
- **`enumerate(iterable)`**：优雅的自动贴标签机。同时返回“索引”和“元素内容”，完美替代 `for i in range(len(lst))`。
- **`len(obj)`**：长度测量仪。获取容器内元素个数。
- **`range(stop)`**：数字序列生成器。

### 二级考试高频处理函数
- **`sorted(iterable, reverse=False)`**：**不伤原数据的排序机**。永远返回一个**新列表**，原数据不变。
- **`zip(*iterables)`**：拉链函数。将多个列表对应位置的元素打包成元组，常用于同时遍历多个序列。

---

## 🧮 4. 数学运算与字符转换

涵盖了日常计算与计算机底层的字符映射。

### 基础与聚合计算
- **`pow(a, b)`**：计算 $$a^b$$，例如 `pow(46, 5)`。
- **`sum() / max() / min()`**：列表数据的总和、最大值、最小值快速计算。
- **`abs(x)`**：绝对值。
- **`divmod(a, b)`**：商余双胞胎。同时返回 `(商, 余数)` 元组，例如 `divmod(10, 3)` 返回 `(3, 1)`。
- **`round(x, d)`**：保留 `d` 位小数。

### 编码与进制魔法
- **`ord(char)` / `chr(int)`**：字符与 Unicode 编码的互逆翻译官。`ord("白")` $\rightarrow$ `30333`，`chr(30333)` $\rightarrow$ `"白"`。
- **`bin() / oct() / hex()`**：将十进制数字分别转换为二进制(`0b`)、八进制(`0o`)、十六进制(`0x`)的**字符串**。
- **`format(value, format_spec)`**：强大的格式化工具，例如 `format(18, "08b")` 将 18 转为 8 位二进制。

---

## 🖨️ 5. 输入输出与自省工具 (NCRE 绝对核心)

### 灵魂剥壳与展示
- **`input(prompt)`**：获取用户输入，**永远返回字符串**。
- **`eval(string)`**：**二级考试的灵魂！** 去掉字符串最外侧的引号，并执行余下的 Python 语句。常配合 `input` 使用：`eval(input())`。
- **`print()`**：基础展示。考试常考 `end=""` 参数以实现不换行输出。
- **`pprint.pprint()`**：优雅打印神器。专门用于复杂字典/列表的换行排版输出（需 `import pprint`）。

### 探索说明书
- **`help(obj)`**：调出内置的官方使用说明文档。
- **`dir(obj)`**：X 光扫描仪。列出对象内部所有的属性和方法。

## 💡 附录：二级考试必背速查表

这里为你总结了最容易混淆和必考的内置函数，考前复习一眼速查。

| **函数名称** | **二级考试核心考点 / 易错点** | **实战示例** |
| :--- | :--- | :--- |
| **`eval()`** | 脱掉字符串外衣，执行代码 | `eval("[1, 2]")` $\rightarrow$ 真实列表 |
| **`divmod()`** | 返回元组 `(商, 余数)` | `divmod(10, 3)` $\rightarrow$ `(3, 1)` |
| **`sorted()`** | 永远返回**新列表**，原数据不变 | `sorted([3, 1, 2])` $\rightarrow$ `[1, 2, 3]` |
| **`zip()`** | 像拉链一样缝合多个列表 | `zip([1], ["a"])` $\rightarrow$ 包含 `(1, "a")` |
| **`bin/oct/hex`**| 返回的是**带有前缀的字符串** | `bin(10)` $\rightarrow$ `"-0b1010"` |
| **`type()`** | 判断数据类型，选择题常考 | `type("白")` $\rightarrow$ `<class 'str'>` |
| **`id()`** | 判断两个变量是否指向同一内存 | `id(a) == id(b)` |

> **复习建议**：重点关注表格中函数的**返回值类型**（是返回元组、字符串还是新列表），这是二级考试选择题最爱设置陷阱的地方。
